# What is this project about?
A completely decentralized platform on the Ethereum blockchain where users can connect with their crypto wallet to view and post messages for other users to see. 

# What tech stack does this project utilise?
Hardhat, Moralis, ReactJS

-- Bootstrapped with React full-stack-dapp sample project and inspiration from Moralis Web3 Twitter Clone project.

Note that this is a toy project for learning purposes.
